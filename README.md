# OpenClaw · A股机构级全维度股票投资分析报告生成技能 V5.0.0

面向A股沪深全市场的机构级投研分析与交易决策Skill，基于**三评分体系**（综合投资价值 / 现价买入性价比 / 交易执行确定性）和**25大模块全链路分析框架**，一键生成标准化机构级投资分析报告。

## 架构总览

```
Layer 5: 输出渲染层  ── HTML三评分看板 · 25模块卡片 · 标准化报告
Layer 4: 报告编排层  ── Jinja2 · 四情景框架 · 证据链校验 · 禁用词过滤
Layer 3: 分析引擎层  ── 25模块分析器 · 三评分引擎 · 联合决策矩阵
Layer 2: 数据处理层  ── AKShare主源 · 多源备援 · 缓存 · 标准化
Layer 1: 接入编排层  ── main.py · 意图识别 · 任务调度 · 并发管理
```

## 快速开始

```bash
pip install -r requirements.txt
python main.py "分析贵州茅台600519"
```

## 数据源

- 主源：AKShare 1.18.64
- 备援：东方财富 / 同花顺 / 新浪财经 / 腾讯财经

## 三评分体系

| 评分 | 含义 | 满分 |
|------|------|------|
| 综合投资价值评分 | 公司质地与长期配置价值 | 100 |
| 现价买入性价比评分 | 当前位置是否适合买入 | 100 |
| 交易执行确定性评分 | 当前机会是否容易执行 | 100 |
